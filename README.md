## _E-Commerce App_
> Karya Lokal is e-commerce app using Turborepo, Clerk, Strapi and Next.js.

### Admin App (CMS)
- Admin Dashboard to view Order.
- Product CRUD.
- Flashsale system.

###  Customer App (SHOP)
- Checkout sytem.
- Cart system.
- Strapi Payment System.
